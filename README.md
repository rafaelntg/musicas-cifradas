# Repositório de músicas cifradas

O objetivo deste repositório é manter as músicas cifradas de maneira cuidadosa, seguindo o padrão de TODAS AS LETRAS MAIÚSCULAS para facilitar a enxergar as letras durante a sua execução.

O posicionamento das cifras segue o mesmo padrão utilizado pelo Cifra Club, ou seja, a cifra deve permanecer sobre a primeira letra da sílaba onde o acorde muda. É importante manter 2 espaços do acorde anterior, pelo menos, para manter a boa visibilidade dos acordes; se necessário, utilize underline ("_") para 
separar uma sílaba da outra.
